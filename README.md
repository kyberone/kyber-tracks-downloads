# kyber-tracks-downloads

This repository serves as the dedicated storage and download source for the music tracks available on [kybertracks.com](https://kybertracks.com).

## Purpose

The audio files hosted within this repository are pulled programmatically and served to users downloading songs through the Kyber Tracks platform. By isolating the audio assets here, it keeps the main website repository lightweight and makes catalog management easier.

## Integration

This repository is primarily accessed by the `kybertracks.com` backend or via direct raw links to serve the downloads.
